# Background Image Required

Please add your `background.webp` image to this directory for the homepage hero section.

## Image Specifications:
- **Filename**: background.webp
- **Recommended Size**: 1920x1080 pixels or higher
- **Format**: WebP (for best performance)
- **Subject**: Barbershop interior, barber tools, or professional barbering scene

## Alternative Formats:
If you don't have a .webp file, you can use:
- background.jpg
- background.png

If using a different format, update the CSS file at line 257:
```css
.hero {
    background-image: url('../images/background.jpg'); /* or .png */
}
```

## Suggested Image Sources:
1. Take a high-quality photo of Steve's Barbershop interior
2. Use professional stock photos from:
   - Unsplash.com
   - Pexels.com
   - Pixabay.com
   
Search terms: "barbershop interior", "barber cutting hair", "classic barbershop"

Once you add the image, the homepage hero section will display it with a beautiful overlay effect.
